# Chris Johannesson – Website

A complete rewrite of my personal website built with modern JavaScript tooling.

🌐 Live site:  
https://chrisjohannesson.com

---

## Overview

This project is a fully client-side web application developed using modern frontend architecture.  
It serves as both a personal profile and a technical reference implementation of a structured React-based website.

The site is implemented as a Single Page Application (SPA) with modular component structure and optimized production builds.

---

## Technology Stack

**Core Technologies**
- JavaScript (ES6+)
- React
- Bootstrap

**Architecture & Tooling**
- Webpack (bundling & build process)
- Babel (transpilation)
- Client-side routing
- Modular component-based structure

**Development Practices**
- Structured project organization
- Separation of presentation and logic
- Responsive design principles
- Production-optimized builds

---

## Project Structure

The application follows a component-driven architecture:

- Reusable React components
- Page-level containers
- Static resource management
- Webpack configuration for development and production environments

The project is built and bundled using Webpack to generate optimized static assets for deployment.

---

## Deployment

The site is deployed as a static frontend application.  
Production builds are generated through Webpack and deployed to the hosting environment serving:

https://chrisjohannesson.com
